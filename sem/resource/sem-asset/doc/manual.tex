\section{会计科目管理}
管理现有的会计科目
    \ul {
    \item  新增会计科目
        \ul {
        \item  点击\button{新建}按钮，进入编辑会计科目界面
        \screenshot{1.png}
        \item  输入\button{编号}及\button{科目名称}
        \item  选择\button{上级科目}（图）
        \item  点击\button{保存}按钮，保存新的会计科目
        }
    \item  查看会计科目
        \ul {
        \item  选择需要查看的会计科目，点击工具条上的\button{查看}按钮，查看会计科目的详细内容
        }
    \item  修改会计科目
        \ul {
        \item  选择需要修改的会计科目，点击工具条上的\button{修改}按钮，进入编辑会计科目界面
        \item  修改需要修改的会计科目明细
        \item  点击\button{保存}按钮，保存更改后的会计科目
        }
    \item  删除会计科目
        \ul {
        \item  选择需要删除的会计科目
        \item  点击工具条上的\button{删除}按钮，在弹出的对话框中点击确定按钮，删除对应会计科目
        \screenshot{2.png}
        }
    }

\section{资产初始化}
初始化资产管理,设置企业出世资产
    \ul {
    \item  新建初始资产单
        \ul {
        \item  点击\button{新建}按钮，进入新建初始资产单界面
        \screenshot{9.png}
        \item  点击\button{添加}按钮，弹出添加初始资产单项目对话框
        \item  选择\button{会计项目}
        \item  选择\button{二级会计项目}
        \item  输入\button{金额}
        \item  选择\button{币种}
        \item  点击\button{确定}按钮，添加初始化资产单项目
        \item  点击\button{保存}按钮，保存初始化资产单
        }
    \item  修改初始资产单
        \ul {
        \item  选择需要修改的初始化资产单
        \item  点击工具栏上的\button{修改}按钮
        \item  修改初始化资产单的相应内容
        \item  点击\button{保存}按钮，保存更改后的初始化资产单
        }
    \item  删除初始资产单
        \ul {
        \item  选择需要删除的初始化资产单
        \item  点击工具栏上的\button{删除}按钮，在弹出的对话框中点击确定，删除对应的初始化资产单
        \screenshot{10.png}
        }
    }

\section{凭证管理}
凭证管理用来管理资金流业务单，实现收支平衡
    \ul {
    \item  新建凭证
        \ul {
        \item  点击\button{新建}按钮进入新建凭证界面
        \screenshot{3.png}
        \item  选择对应的资金流业务单
        \item  点击\button{添加}按钮添加凭证项目，确保借贷平衡
        \screenshot{4.png}
        \item  点击\button{保存}按钮，保存新凭证
        }
    \item  修改凭证
        \ul {
        \item  选择需要修改的凭证，单击\button{修改}按钮
        \item  修改需要修改的项目或对应的资金流业务单
        \item  点击\button{保存}按钮，保存修改后的凭证
        }
    \item  删除凭证
        \ul {
        \item  选择需要删除的凭证，点击\button{删除}按钮
        \item  在弹出的对话框中点击\button{确定}按钮，删除凭证
        \screenshot{5.png}
        }
    }
